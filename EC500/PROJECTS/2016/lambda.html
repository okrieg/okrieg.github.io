Project Overview

Amazon’s Lambda product (https://aws.amazon.com/lambda/) offers a compelling model for deploying and running code in response to events in the Cloud with zero infrastructure. It’s a compelling model for all kinds of integration and computation tasks that run in response to events/messages/streams. The team would develop an equivalent capability that could be deployed on an OpenStack cloud like the MOC. The capability should be packaged so that it can be deployed onto other OpenStack clouds, maybe published in something like the Murano app catalog.

    What is CONS3RT?

    CONS3RT is a leading-edge hybrid cloud orchestration and service management software suite, enabling automated, user-driven DevOps workflows. There are several tools that target cloud management, but CONS3RT focuses on cloud usefulness so users can get things done! Users can leverage the Provisioning as a Service (PraaS), Build as a Service (BaaS), and Test as a Service (TaaS) capabilities so their organization can achieve the highest levels of continuous integration and continuous delivery.

    The CONS3RT framework allows for the development of adapters for different cloud, provisioning and test technologies. The objective of this project is to develop a CONS3RT-to-MOC adapter, deploy it and connect to a CONS3RT site. The team will work within an existing development effort and leverage a robust test and development process to support the project. Team members will design and execute DevOps workflows to reach "done".

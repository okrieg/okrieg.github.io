Project Overview

The increase in telecommuting and other flexible work environments requires better connectivity over disparate locations. We will build a network solution that allows workers to connect to each other via the cloud in a way that all appear to be on a virtual switch in a local network. This will require securing access with OpenID Connect and providing connectivity between Windows/OSX/Linux.
